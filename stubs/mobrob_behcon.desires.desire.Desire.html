The class Desire

A desire consist of

  • a value (any number/array),

  • a dynamic strength (0.0 - 1.0) and

  • a static priority (0 - 100)

  • a function to be executed if desire is accepted (optional)

The priority will be set when Desire is forwarded to resolver.
